== Troubleshooting

*Can't log in?*
→ Make sure you copied the credentials exactly. The API URL is `{openshift_api_url}`.

*Pod not starting?*
→ Run `oc describe pod <pod-name> -n <namespace>` to see events and error details.

*Route URL not loading?*
→ Allow 30-60 seconds after route creation. Routes become active asynchronously.

*Wrong namespace?*
→ Run `oc project` to check your current namespace. Use `oc project <namespace>` to switch.
